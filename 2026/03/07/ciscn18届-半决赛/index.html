漏洞点在edit

int snprintf(char *str, size_t size, const char *format, …);

1 目标缓冲区地址
2 最大写入长度
3 格式字符串
4 后面的格式化参数

在 glibc 中：

参数顺序是：

参数 含义
str 输出缓冲区
size 最大写入长度
format 格式化字符串
… 格式化参数

eg: snprintf(buf, 0x20, "%lu", value);
